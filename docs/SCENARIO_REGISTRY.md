# Scenario Registry

The `Data/...` paths below are local generation output roots. They are not bundled in the GitHub code repository; hosted benchmark payload files are distributed through Hugging Face.

| Paper name | Hugging Face subset | Local output root when generated | Generator module | Implemented mechanism | Transfer | Notes |
|---|---|---|---|---|---:|---|
| Reference | reference | Data/SynLV/scenario_A | benchmark_generation/primary_scenario_generation.py | Reference cohort with minimal natural missingness | false | Canonical reference cohort from Data/SynLV. |
| MNAR Mild | missing_not_at_random_mild | Data/SynLV/scenario_B | benchmark_generation/primary_scenario_generation.py | Independent MNAR missingness with alpha=0.8, latent severity, and event-only proximity | false | The code proximity term is applied only for event patients. |
| MNAR Strong | missing_not_at_random_strong | Data/SynLV/scenario_C | benchmark_generation/primary_scenario_generation.py | Independent MNAR missingness with alpha=1.6, latent severity, and event-only proximity | false | The same event-only proximity term is used with larger alpha. |
| MNAR Correlated | missing_not_at_random_corr | Data/SynLV_Zoo/base/scenario_MNAR_CORRELATED | benchmark_generation/generate_synlv_variant_zoo.py | MNAR Mild-like marginals with Gaussian-copula cross-channel mask dependence | false | Generated by the variant-zoo release module. |
| MAR | missing_at_random | Data/SynLV/scenario_MAR | benchmark_generation/primary_scenario_generation.py | MAR missingness depending on generated categorical covariates | false | MAR masks depend on observed categorical features. |
| VisitShift | visitshift | Data/SynLV/scenario_VISITSHIFT | benchmark_generation/primary_scenario_generation.py | Visit-schedule control under the base visit sampler | false | Visit-schedule control under the base visit sampler. |
| VisitShift-Transfer | visitshift_transfer | Data/SynLV_Zoo/base/scenario_VISITSHIFT_TRANSFER | benchmark_generation/generate_synlv_variant_zoo.py | Reference train/validation with VisitShift test source | true | Transfer wrapper offsets test RANDID values to avoid split overlap. |
| Mismatch | mismatch | Data/SynLV/scenario_MISMATCH | benchmark_generation/primary_scenario_generation.py | Alternative MNAR observation regime with informative missingness and alpha=0.8 | false | Alternative MNAR observation regime with informative missingness. |
| Mismatch-Transfer | mismatch_transfer | Data/SynLV_Zoo/base/scenario_MISMATCH_TRANSFER | benchmark_generation/generate_synlv_variant_zoo.py | Reference train/validation with Mismatch test source | true | Test source inherits the Mismatch MNAR observation regime. |
