# MIMIC Grounding Boundary

MIMIC is used as a grounding layer only.

This code repository and the hosted SynLV synthetic benchmark release do **not** redistribute:
- raw MIMIC data,
- row-level derived MIMIC tables,
- patient-level extracted MIMIC intermediates.

What is included:
- reconstruction-oriented code paths,
- manifests/notes needed to reproduce the grounding workflow,
- benchmark-level documentation.

Rebuilding MIMIC grounding results requires credentialed source access under PhysioNet/MIMIC terms.
